## **② 游戏与曲子背景**

哈喽大家好 我是配乐人琳发财

快过年了最近是越来越冷了

这种冰冷的日子里，我们来做一首让人热血沸腾的曲子吧

**《空洞骑士：丝之歌》（Hollow Knight: Silksong）** 是 Team Cherry 开发的横版探索游戏。

这一次主角从骑士换成了 **黄蜂（Hornet）**，整个世界的氛围比前作更快、更灵动、更带“丝线”那种锐利的质感。

游戏延续了前作的暗黑童话风格，但整体动作更轻盈，敌人机制更多，也更强调节奏感和瞬间反应，所以音乐必须跟得上这种“高速战斗”的世界观。

作曲家依然是前作的作者 —— **Christopher Larkin**。

他的音乐特点是：

- 擅长用弦乐营造“紧张又优雅”的情绪
- 战斗曲里会大量使用断奏、变拍子，让音乐像在呼吸、像在奔跑

**蕾丝战斗曲（Lace Boss Theme）** 就是 Larkin 在丝之歌里最典型的一段战斗音乐。

角色蕾丝（Lace）动作极快、攻击带刺、节奏突然又锐利，所以整首曲子完全用弦乐写成

**密集、优雅、致命、像一把带丝线的细剑。**

这首曲子本身没有旋律主角，而是靠弦乐不断“蓄力、推进、刺击”来构成战斗气氛，是丝之歌音乐风格最具代表性的段落之一

## **③ 全曲结构总览（1 分钟）**

整首曲子时长2分40秒

分为6段，intro，A，B，C，D，Ending

BPM大概是153左右

节拍比较复杂，5/4拍为主，穿插一些3/4拍，6/4拍和6/8拍

正常的4/4拍会让动作很均匀，但一旦变成5/4或6/4，音乐就会变成‘长一下’或者‘短一下’，这种不规则的步伐非常适合表现敌人攻击的变化节奏，也让战斗氛围更真实

变拍子的效果就像战斗的节奏永远不稳定，你永远猜不到下一步，是前冲、是停顿、还是突然换招。所以音乐会让人觉得特别紧张、特别有推进感，完全没有‘数拍子数到麻木’的感觉

配器上非常简单，是纯弦乐

我这次使用的音源是好莱坞弦乐第二代

## **④ 分段和声进行分析**

和声分析之前我要说明一下，我写的和声并非百分百正确

大概意思没有错，但具体和声有些不一定

因为好些小节是双音，有的我猜的，有的就空了

但是不影响我们理解它的和声进行

Intro：Gmin → B♭maj7/13 → D/F# → E♭ → Gmin → B♭maj7 → G♭aug → E♭ → E♭♯11/D → D

Intro 一上来就是 G 小调，但它没有乖乖走常规路线。和弦不断往外拐，尤其是那个增三和弦，让音乐像被突然扭了一下，紧张感瞬间拉满。整体就是‘要落地但又被拉走’的感觉，让战斗的氛围从第一秒就冲出来

A 段： 

Gmin → E♭ → Cmin → Gmin → E♭ → Cmin → D →

Gmin → B♭6 → E♭ → Gmin → E♭maj7/B♭ → E♭maj7/G♭ → E♭

A 段基本是在 G 小调里，用 G 小调和它的关系和弦 C 小调来反复堆叠动能。

其中夹着降E 和弦，给音乐带出一点“明亮但忧伤”的拉扯感。

它的推进方式不是靠大跳，而是靠同一组小调和弦的反复堆叠，这种“压着往前推”的写法，让断奏特别有冲劲。

B段：

Gmin → B♭aug → E♭maj7 → Gmin9/F → C7 → D → F → Cmin7 → Gmin →

E♭maj7 → Gmin/D → C7 → Gmin → D → E♭maj7 → E♭maj7/F → F♯dim

B 段还是在 G 小调，但气质跟 A 段不一样。它开始加入更多色彩和弦，比如增三和弦、带九和弦的 G 小调，还有 C7 这种突然亮一下的属和弦，让音乐的紧张感变得更‘旋转’、更不稳定。整体感觉是：重心还在小调，但线条变得更灵活，像是在战斗中不断变换姿势

C段：

Gmin → Gmin → Dsus4→Gmin → Gmin → Gmin → A7 → Gmin → B♭maj7 → F♯min → E♭ →

Gmin → B♭6 → B♭maj7 → G♭aug → E♭min → D♯dim → D

C 段还是在 G 小调里，但情绪比前面更压抑、更拉扯。它用大量重复的 G 小调和突然出现的 A7、增三和弦、半音下行的色彩和弦，把战斗的紧张感拉到极致。最后用一个回到 D 的属和弦，把下一段的爆发感吊住了

这一段的特色是 ‘明明在原地踩，却一直往下沉’——反复回到 G 小调，让情绪越堆越重；中间混入 A7、B♭maj7、增三和弦和减和弦，让色彩不断扭曲，像战斗中被持续压迫一样

D段：

Gmin → B♭aug → E♭maj7 → F/D → Cmin → D → F → C →

Gmin → G♭aug → E♭maj7 → F6 →

Gsus4 → Gmin → D → E♭maj7 → D →

C → Gmin → E♭ → Cmin

D 段其实是 B 段的强化版，一样在 G 小调里不断扭、不断推，但这次加入更多明亮的跳转，比如 F、C、E♭maj7 这些和弦，让整段像是战斗中的‘第二波爆发’。增三和弦再次出现，把紧张的扭曲感推到最顶。

Ending：

Dsus4 → D → E♭ → F7 → Fmin7 → A♭ → Gmin

Ending 前半段用了很多亮色和弦来把情绪推到顶点，接着是一段大幅度的弦乐滑音，像能量完全甩出去一样。但真正的结尾并不是滑音，而是最后稳稳落在一个 G 小调上，用最悲壮的方式把整首曲子收住

然后我们分段讲一下配器

## **⑤ 分段配器分析**

Intro：

**高弦（Vln 1 & Vln 2）——负责攻击、节奏、锐度**

- 做“刀锋感”
- 模拟战斗的快速剑击 / 连击动作
- 给 Intro 打下最锋利的节奏线条

**中弦（Viola）——负责连贯、推进、稳定中频**

- 把音乐“往前推”
- 填满中频，让高弦不会显得干
- 增强战斗的“持续性”

 **低弦（Cello & Bass）——负责重心、推进的落点**

- 打“地面感”
- 决定每小节的重心
- 把整个 5/4 拍推得更稳、更紧张

Intro 是典型的丝之歌式室内乐战斗写法：高弦就是这段的攻击线，走主旋律，中提琴负责推进，给一个律动，低弦负责重心，像战斗的脚步，每个落点告诉你节奏怎么走。三层咬合得非常紧，所以即便是纯弦乐，战斗感仍然爆棚

A：

 **开头：高弦持续“攻击感”（刺击型写法）**

- 建立“近战、快速刺击”的感觉
- 是弦乐战斗曲最典型的攻击线条

 **中间两小节：突然“蓄力”**

这是丝之歌常用“战斗转折”写法。

- **把能量压缩起来**
- 让后面的节奏感显得更爆

 **蓄力结束后：节拍从 3/4 → 5/4，重新启动推进**

- 听感变得更稳、更推进
- 战斗从混乱 → 重新进入节奏
- 像人物从“爆发动作”回到连击阶段

A段高弦像一开始连续挥出的刀光，节奏密度很高，战斗马上开始。

中间两小节在做蓄力动作，整个弦乐组齐刷刷往上推，你可以想象蕾丝换阶段攻击

蓄力结束后节拍切回 5/4，音乐又重新开始往前冲，这是战斗节奏重启的信号

总结就是攻击 → 蓄力 → 推进

B：

**大提琴先给长线条：把战斗的“地面”铺开**

- 扩大声场
- 拉开空间
- 暂时解除前一段的密度压迫

 **小提琴群中段接手长线条：从宽场景 → 抬头准备下一次攻击**

- 把注意力从低区移到高区
- 情绪变得更“焦灼，但不是爆发”
- 进入过渡区的典型写法

 **段尾：节奏回到密集，重新启动战斗推进**

B段大提琴先放出大面积的长音，就像把战斗的地面铺开，把气氛拉宽，让紧张的节奏先喘一口气

中段开始换成小提琴接力长音，音乐像从地面抬到空中，情绪慢慢往上升，为下一次攻击做准备

段尾节奏一点点加回来，弦乐密度提升，战斗重新启动，准备回到主循环

C：

**C 段几乎没有“旋律”，只有“方向”**

- 1/2 提琴的线条是持续性的上行
- 多声部叠合成“上升坡”
- 没有 hook、没有主题、没有动机重复

**三层“上升”同时发生，形成巨大压力**

- **高弦：短音 → 密集上行**（推动空气升高）
- **中弦：稳定节奏脉冲**（保持肌肉紧绷）
- **低弦：断奏向上挑**（底盘也在加压）

**变拍子让上升变得不规则，更危险**

- 身体感上的失衡
- 上升过程变得更不稳定
- 压迫感更强

C 段的写法是典型的蓄力段，没有主旋律，也没有明显的动机重复，而是让弦乐分部做持续的上行、叠加、扩张。高弦在往上推，中提琴在往上推，大提琴也在往上推，每一个声部都同时在累积能量。

因为是不断往上走的线条，所以听感上会有一种‘战斗要爆发了’的紧张感

D：

**中提琴先铺地：更宽、更稳、更成熟的“长线条开场”**

- 线条范围更高
- 声场更亮
- 情绪更“昂起来”而不是“沉下去”

**一提给复调：复杂度提升、战斗感升级**

- 小范围模仿
- 节奏错落
- 不是旋律，而是“能量的搅动”

**一提 + 二提的长线条联手：气势直接拉满**

- 高弦、中弦**一起往上拉长音**
- 范围更广
- 方向更明确（都是上升）
- 能量比 B 段强得多

**尾巴回到 A 段经典“刺击型”：让段落重启、回到战斗节奏**

- 直接引用 A 段“刺击型高弦”
- 但这次密度更高
- 更短、更狠
- 是“重启战斗”的信号

D 段是 B 段的升级版：中提琴先铺出更宽的长线条，一提加入复调让段落变得更复杂，之后高弦和中弦一起往上拉出大幅度的长音，把气势抬满。最后突然回到 A 段开头那种密集刺击，像把蓄好的能量瞬间引爆

Ending：

**开头：最后一次蓄力，把情绪推到最顶点**

- 高弦密集短音往上爬
- 中提琴同步跟上
- 大提琴也进行短音加速

**中段：滑音下降＝突然抽掉地心引力**

- 所有弦乐一齐向下滑
- 像坠落
- 很快
- 很整齐

**滑音后的二次上升：比前面更亮、更决绝**

- 这次不碎
- 是更明确的长线条
- 往更高音区冲
- 范围更大

**最终收尾：厚、重、华丽、像锁门一样的 ending**

- 齐奏
- 落点很重
- 和声定位清晰
- 给人一种“最终定格”的重量感

Ending 的写法非常漂亮：先是一段持续上升的蓄力，把能量整个托到最高点；接着弦乐做了一个快速滑音，像是把空气瞬间抽掉；再往上拉一段更亮、更强的上升线条，最后用几下非常重、非常华丽的收尾，把整首战斗曲彻底锁死
